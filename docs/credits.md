# ClassBridge — Credits

## Built By

ClassBridge was built for **TechCommons Hacks V1** — Global & Local Impact theme, Education & Accessibility track.

## Technologies

- [Next.js](https://nextjs.org/) — React framework
- [React](https://react.dev/) — UI library
- [TypeScript](https://www.typescriptlang.org/) — Type safety
- [Tailwind CSS](https://tailwindcss.com/) — Utility-first CSS
- [shadcn/ui](https://ui.shadcn.com/) — Component library
- [Zod](https://zod.dev/) — Schema validation
- [NVIDIA NIM](https://build.nvidia.com/) — AI inference API
- [Lucide Icons](https://lucide.dev/) — Icon library

## AI Models

- **LLaMA 3.1 8B Instruct** via NVIDIA NIM — Used for all AI inference including content extraction, simplification, translation, study card generation, practice question creation, and parent/teacher summaries.

## Design Inspiration

- Warm, student-friendly design language
- Education-focused color palette (calm blue/teal)
- Accessible typography with large, readable text
- Card-based layout for study materials
